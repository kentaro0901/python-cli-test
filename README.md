# python-cli-test

## 概要
pythonでCLIアプリケーションを作ったテスト

## 作成コマンド
```bash
$ python setup.py develop # ローカルにインストール
$ python setup.py sdist # 配布用パッケージの作成
$ pip install https://${host}/samplecli-0.0.1.tar.gz # パッケージからインストール
```
